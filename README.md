# The Guild
<img width="624" alt="guild-1" src="https://user-images.githubusercontent.com/61943384/120079877-f196d000-c0ad-11eb-87a2-8fed68cd845d.png">
The Guild is a social media platform for families, groups of friends, gaming clans or even project groups and co-workers. Allows you to quickly communicate with each other and
work together towards a common goal through milestones and tasks.

Interested?

[Sign up and have a look!](https://theguild-lw.herokuapp.com)

# Features

### Guild list page
A guild is a where a medium or large group of people can create posts, organise events, share accomplishments and things important, set milestones for activities and keep track of the tasks.

<img width="624" alt="guild-2" src="https://user-images.githubusercontent.com/61943384/120080213-be554080-c0af-11eb-9c13-c0ccacb7acca.png">

### Guild create page

<img width="624" alt="guild-3" src="https://user-images.githubusercontent.com/61943384/120080221-c4e3b800-c0af-11eb-825d-550bafd1f695.png">

### Guild page
- A user can create, delete and edit a post on the guild page.
- A user can see only the posts with pictures.
- A user can see all guild members and check member's status(an admin user has a crown on a user icon).

<img width="624" alt="Screenshot 2021-05-29 at 21 16 38" src="https://user-images.githubusercontent.com/61943384/120083805-37aa5e80-c0c3-11eb-818c-260b6a129c32.png">

### Guild manage page
- An admin user can create a milestone.
- An admin user can manage, delete and edit the guild.
- An admin user can assign other people as admins or ban them.
- An admin user can invite other people to join.

<img width="624" alt="guild-5" src="https://user-images.githubusercontent.com/61943384/120080331-43d8f080-c0b0-11eb-9223-837f21048d33.png">

<img width="624" alt="guild-6" src="https://user-images.githubusercontent.com/61943384/120080333-4afffe80-c0b0-11eb-985b-67e5ac066b29.png">

### Milestone list page
- A user can see all the active milestones in an active milestone list page.

<img width="624" alt="Screenshot 2021-05-29 at 21 14 55" src="https://user-images.githubusercontent.com/61943384/120083764-f87c0d80-c0c2-11eb-9374-0e2e888ad65e.png">

### Milestone create page
A milestone is a where guild members can organise tasks, see the progress in a glance and set a reward to enjoy when all tasks are completed.
- An admin user can create a milestone.

<img width="624" alt="guild-7" src="https://user-images.githubusercontent.com/61943384/120080337-4fc4b280-c0b0-11eb-8db4-71ff9a6684e2.png">

### Milestone page
- A user can activate or destroy a milestone.
- A user can tick a task as done.

<img width="624" alt="guild-8" src="https://user-images.githubusercontent.com/61943384/120080342-57845700-c0b0-11eb-8eb6-4ded8f0ca0cf.png">

### Create a new task
- A user can create a new task and assign a member.
